Hola que tal a todos, los pasos para ejecutar el sistema son los siguientes
1. Tener MySQL y Node js instalados (Tener la contrase침a de tu MySQL si no usar XAMMP o algo asi creo se llama xD)
2. Ejeutar en un entorno MySQL los archivos adjuntos a este archivo:
    . Base de datos a usar
    Admin view - FIXED
    Scheduller
    Triggers
3. Una vez ejecutado los scripts de MySQL dirijirse a la carpeta backend y ejecutar
  cd backend
  npm init -y
  npm install
  Instalara las dependencias
4. Una vez ejecutado en la raiz de la carpeta crear los siguientes archivos:
    1. .env con lo siguiente:
    JWT_SECRET=tu_clave_secreta_muy_segura_123
    DB_HOST=localhost
    DB_USER=root
    DB_PASSWORD=TuContraMySQL
    DB_NAME=academia_final
    PORT=4000

    2. db.js con lo siguiente:
    // db.js
    const mysql = require('mysql2/promise');

    const connection = mysql.createPool({
      host: 'localhost',
      user: 'root',
      password: 'contrase침a de tu BD',
      database: 'academia_final',
      waitForConnections: true,
      connectionLimit: 10,
      queueLimit: 0
    });

    // Verificar la conexi칩n
    connection.getConnection()
        .then(conn => {
            console.log('Connected to the MySQL database.');
            conn.release();
        })
        .catch(err => {
            console.error('Error connecting to the database:', err);
        });

    module.exports = connection;

3. Crear la carpeta uploads en la raiz de la carpeta backend esto para la subida de imgenes

4. Si tienes fe al momento de ejecutar node server.js te correra

5. Si no ejecutar los scripts de la carpeta test que fueron generadas por Cursor (recomendable descargar ese idle)

        cd scripts
        node createAdmin.js

      (Por el momento solo se puede subir un voucher, 
      al momento de matricular te sale a que cuenta pagar, modificar si el cliente lo quiere)
